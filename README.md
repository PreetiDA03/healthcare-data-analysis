# healthcare-data-analysis
Healthcare Data Analysis using Python (Pandas, Seaborn, Matplotlib)


## 📊 Project Overview
This project analyzes healthcare data to understand patient trends, medical conditions, and billing patterns.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Analysis Performed
- Data Cleaning
- Feature Engineering (Stay Days)
- Descriptive Analysis
- Financial Analysis
- Time Series Analysis
- Data Visualization

## 📈 Key Insights
- Patient distribution is balanced across medical conditions
- Emergency admissions tend to have higher billing
- Healthcare demand is consistent across months
- Billing varies based on medical condition and stay duration
